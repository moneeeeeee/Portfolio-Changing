Automated Single-Cell Aspiration System

Electrical and Software Systems Lead | Arduino, MATLAB, Fritzing, SolidWorks

Overview

I led control system design for a microfluidic device that isolates single cells using negative pressure, with closed-loop feedback to maintain target suction.

Tools

  • Arduino Uno, stepper motor control
  • MPX5010 pressure sensor, INA125 instrumentation amplifier
  • MATLAB for early logic tests
  • Fritzing for circuit layout, SolidWorks for fixture design

Role

Electrical and Software Systems Lead

Description

I built a feedback loop that compares live pressure to a potentiometer setpoint and adjusts motor steps to keep suction within ±0.5 kPa. I streamed live data to Excel for monitoring and tuned control gains for stable response. I integrated valve logic with the biological loading phase and produced wiring diagrams and test logs for team handoff.

The system supports single-cell sequencing and screening workflows where precise, repeatable handling is essential. The project strengthened my skills in embedded control, sensor conditioning, and lab integration.
